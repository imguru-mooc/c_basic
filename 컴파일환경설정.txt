
무료 컴파일러 설치 

1. 아래 사이트 접속
https://visualstudio.microsoft.com/ko/vs/features/cplusplus/

2. Visual Studio에서 C++ 사용 >> Community 2019 선택

3. 다운로드 된 아래 파일 실행
  vs_community__rid.30005.exe

4. 설치 버튼 클릭


프로젝트 만들기 

1. 파일 >> 새로만들기 >> 프로젝트 선택

2. 빈 프로젝트 선택 >> 다음 클릭

3. 프로젝트 이름 입력 >> 만들기 클릭

소스 파일 추가 

1. 솔루션 탐색기 >> 소스 파일 오른쪽 버튼 클릭

2. 추가 >> 새항목 

3. 이름 : 파일이름.c 입력 >> 추가

컴파일 및 실행 

컴파일 
빌드 >> 솔루션 빌드 

실행 
디버그 >> 디버그 하지 않고 시작 선택
